<h1 align="center">Chat Real Time</h1>
<p align="center">The project was inspired by the slack, where users have several channels with which they can interact</p>

![](chatRealTime.gif)

### Prerequisites

Before you begin, you will need to have the following tools installed on your machine:
[Git](https://git-scm.com), [Node.js](https://nodejs.org/en/), [Yarn](https://yarnpkg.com/). 
In addition, it is good to have an editor to work with the code. I recommend the [VSCode](https://code.visualstudio.com/)

### üé≤ Running The Backend (socket server)

```bash
# Clone this repository
$ git clone https://github.com/sam20fonsa1098/ChatRealTime.git

# Access the project folder in the terminal
$ cd ChatRealTime

# Go to backend folder
$ cd backend

# Install the dependencies
$ yarn install
```
